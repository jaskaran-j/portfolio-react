# Jaskaran.dev — Personal Portfolio

Personal portfolio website built with React and Vite.

🔗 **Live site:** https://jaskarandeep.me

## About
Final year Computer Science student interested in data, machine learning, and cybersecurity. This portfolio showcases my projects, experience, and skills.

## Tech Stack
- React
- Vite
- React Router
- CSS
- Deployed on Vercel
- Custom domain via Namecheap

## Pages
- **Home** — Introduction and tech stack
- **About** — Background, education and interests
- **Projects** — Showcase of work
- **Experience** — Work, volunteering and competitions
- **Contact** — Ways to get in touch

## Run locally
```bash
git clone https://github.com/jaskaran-j/portfolio-react.git
cd portfolio-react
npm install
npm run dev
```
